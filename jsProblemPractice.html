How Window.print() Method Work

window.print()

This Method prints the contents of the current window.The print() method opens the Print Dialog Box, which lets the user to select preferred printing options.

JavaScript toLocaleDateString() Method

toLocaleDateString()

The toLocaleDateString() method converts the date (not the time) of a Date object into a readable string, using locale conventions.

JavaScript date Method

  • getDate()

  • getMonth()

  • getFullYear()

  • The getDate() method returns the day of the month (from 1-31) for the specified date.

  • The getFullYear() method returns the year of the specified date.

  • The getMonth() method returns the month (from 0 to 11) for the specified date, according to local time.
